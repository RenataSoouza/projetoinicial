### 📖 Sobre
Este projeto marca o meu primeiro passo na jornada de desenvolvimento, onde explorei a criação de hipertexto para compartilhar informações de maneira eficaz e envolvente. Neste contexto, meu objetivo principal foi criar um conteúdo que pudesse ser facilmente compreendido pelos leitores e proporcionar uma experiência informativa e agradável.
